# Sphere

**Namespace:** `MATHLIB` &nbsp;·&nbsp; **Library:** Math Library

:::info[Python equivalent]
`mne.make_sphere_model` in MNE-Python.
:::

`#include <math/sphere.h>`

```cpp
class UTILSLIB::Sphere
```

Best-fit sphere primitive carrying centre and radius, with static factories for the closed-form algebraic fit (Jennings) and the Nelder–Mead geometric-residual refinement.

Used to build spherical head models, fit HPI coils and digitiser scalps.

3-D sphere value type with algebraic and Nelder–Mead best-fit factories.
