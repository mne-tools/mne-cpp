# Warp

**Namespace:** `MATHLIB` &nbsp;·&nbsp; **Library:** Math Library &nbsp;·&nbsp; **Header:** `math/warp.h`

`#include <math/warp.h>`

```cpp
class UTILSLIB::Warp
```

Thin-plate-spline (Bookstein) 3-D warp that maps source landmarks exactly onto destination landmarks and extends smoothly to the rest of `R^3`.

Used for non-rigid registration of EEG/MEG sensors and template-to-subject mesh warping.

Thin-plate-spline 3-D warp fitted from landmark correspondences.
