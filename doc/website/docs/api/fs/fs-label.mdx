# FsLabel

:::info[Python equivalent]
`mne.read_label` in MNE-Python.
:::

`#include <fs/fs_label.h>`

```cpp
class FSLIB::FsLabel
```

A FreeSurfer/MNE surface label: per-vertex indices, Tk-RAS positions and scalar values for one hemisphere.

Wraps the parsed contents of a ``.label file or an in-memory label constructed programmatically (e.g. by thresholding a source estimate via `FsLabelUtils::stcToLabel`). Vertex indices reference the matching `FsSurface` for the same subject and hemisphere; positions are the Tk-surface RAS coordinates copied from that surface; values are the per-vertex scalar payload (statistic, time, weight). Hemisphere is encoded with the FreeSurfer convention \{0 = lh, 1 = rh\}.
