# RtInvOp

**Namespace:** `RTPROCESSINGLIB` &nbsp;·&nbsp; **Library:** DSP Library

`#include <dsp/rt_inv_op.h>`

Controller that manages `RtInvOpWorker` for online inverse operator updates.
