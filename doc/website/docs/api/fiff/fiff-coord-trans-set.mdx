# FiffCoordTransSet

**Namespace:** `FIFFLIB` &nbsp;·&nbsp; **Library:** FIFF Library

`#include <fiff/fiff_coord_trans_set.h>`

```cpp
class FIFFLIB::FiffCoordTransSet
```

Container for the FIFF coordinate transforms found in (or written to) a FIFFB_MRI / FIFFB_HPI_MEAS block.

Owns the list of `FiffCoordTrans` and the metadata that travels with them (fiducial points, registration comments). Provides lookup by (`from`, `to`) frame pair so consumers can ask for, e.g., `FIFFV_COORD_HEAD` → `FIFFV_COORD_MRI` without iterating manually.
