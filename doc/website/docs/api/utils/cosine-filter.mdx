### CosineFilter(fftLength, lowpass, lowpass_width, highpass, highpass_width, sFreq, type)

Constructs a `CosineFilter` object.

**Parameters:**

- **fftLength** : *int*
  length of the fft (multiple integer of 2^x).

- **lowpass** : *float*
  low cutoff frequency in Hz (not normed to sampling freq).

- **lowpass_width** : *float*
  determines the width of the filter slopes (steepness) in Hz (not normed to sampling freq).

- **highpass** : *float*
  highpass high cutoff frequency in Hz (not normed to sampling freq).

- **highpass_width** : *float*
  determines the width of the filter slopes (steepness) in Hz (not normed to sampling freq).

- **sFreq** : *double*
  sampling frequency.

- **type** : *TPassType*
  filter type (lowpass, highpass, etc.).
