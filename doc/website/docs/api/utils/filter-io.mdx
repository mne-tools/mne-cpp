### readFilter(path, filter)

Reads a given txt file and scans it for filter coefficients.

Pls see sample file for file syntax.

**Parameters:**

- **path** : *QString*
  holds the file path of the txt file which is to be read.

- **filter** : *FilterKernel &*
  holds the filter which the read parameters are to be saved to.

**Returns:**

- *bool* — true if reading was successful, false otherwise.
