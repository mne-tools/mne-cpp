### evalMatrixViaFile(code, outputFilePath, ok, timeoutMs)

Run Python code that writes a matrix to outputFilePath (e.g.

via numpy.savetxt), then read the result back as an Eigen MatrixXd.

This is preferred over evalMatrix() for large matrices or when full double precision is required, because it avoids stdout parsing.

Example: 
```cpp
QString code = QString(
    "import numpy as np\n"
    "cov = np.eye(3)\n"
    "np.savetxt('%1', cov, fmt='%%.17e')\n"
).arg(outPath);
bool ok;
auto mat = helper.evalMatrixViaFile(code, outPath, &ok);
```

**Parameters:**

- **code** : *const QString &*
  Python code to execute.

- **outputFilePath** : *const QString &*
  Path where Python writes the matrix.

- **ok** : *bool **
  Set to true on success.

- **timeoutMs** : *int*
  Timeout in milliseconds.

**Returns:**

- *Eigen::MatrixXd* — Parsed MatrixXd, or empty matrix on failure.

### isPythonRequired()

Check whether the environment demands Python availability.

When MNE_REQUIRE_PYTHON=true is set, test guards should QFAIL instead of QSKIP, ensuring skips are never silent in CI.

**Returns:**

- *bool* — True if MNE_REQUIRE_PYTHON environment variable is "true" (or "1").

---

### writeMatrix(filePath, mat)

Write an Eigen matrix to a text file with full double precision.

The format is one row per line, space-separated values, using %.17e notation — compatible with Python's numpy.loadtxt().

**Parameters:**

- **filePath** : *const QString &*
  Output file path.

- **mat** : *const Eigen::MatrixXd &*
  Matrix to write.

**Returns:**

- *bool* — True on success.

---

### readMatrix(filePath, ok)

Read an Eigen matrix from a text file (space-separated values, one row per line).

Compatible with numpy.savetxt() output.

**Parameters:**

- **filePath** : *const QString &*
  Input file path.

- **ok** : *bool **
  Set to true on success, false on failure.

**Returns:**

- *Eigen::MatrixXd* — Parsed MatrixXd, or empty matrix on failure.
