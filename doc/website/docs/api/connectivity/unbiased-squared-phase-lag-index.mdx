# UnbiasedSquaredPhaseLagIndex

**Namespace:** `CONNECTIVITYLIB` &nbsp;·&nbsp; **Library:** Connectivity Library

`#include <connectivity/unbiasedsquaredphaselagindex.h>`

```cpp
class CONNECTIVITYLIB::UnbiasedSquaredPhaseLagIndex
```

Computes the unbiased estimator of squared PLI for every channel pair.

The estimator subtracts the analytically known diagonal contribution from the squared sum of cross-spectral sign samples, yielding an estimate whose expectation equals `PLI^2` for any trial count `N` &gt;= 2 and is therefore free of the upward small-sample bias that affects the plain `PhaseLagIndex`. The construction preserves the zero-lag rejection property of the PLI family.

Unbiased squared Phase Lag Index estimator; removes the small-sample bias of `PhaseLagIndex`.
