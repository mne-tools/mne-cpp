# CovarianceSettingsView

**Namespace:** `DISPLIB` &nbsp;·&nbsp; **Library:** Display Library &nbsp;·&nbsp; **Header:** `disp/viewers/covariancesettingsview.h`

`#include <disp/covariancesettingsview.h>`

```cpp
class DISPLIB::CovarianceSettingsView
```

Settings panel exposing the noise-covariance estimation window length.

Holds a single `QSpinBox` forwarded as `changeMinNumberSamples` to the `rtprocessing` covariance estimator.
