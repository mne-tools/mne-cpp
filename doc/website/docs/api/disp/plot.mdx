# Plot

**Namespace:** `DISPLIB` &nbsp;·&nbsp; **Library:** Display Library &nbsp;·&nbsp; **Header:** `disp/plots/plot.h`

`#include <disp/plot.h>`

```cpp
class DISPLIB::Plot
```

MATLAB-style line plot for an Eigen vector, derived from `Graph`.

Owns a list of `QVector<QPointF>` trace paths and supports a `m_bHoldOn` flag for overlaying multiple curves. ``updateData()`` replaces the visible curve in place.
