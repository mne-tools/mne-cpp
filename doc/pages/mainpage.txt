/*! \mainpage MNE-CPP (MNE1) Documentation
 *
 * \section intro_sec Introduction
 *
 * Fast MNE Qt 5 based C++ library (beta)
 *
 * \section get_the_code_sec Getting the latest code
 *
 * To get the latest code using git, simply type:
 *
 *    git clone git://github.com/mne-tools/mne-cpp.git
 *
 * If you don't have git installed, you can download a zip or tarball
 * of the latest code: <a href="https://github.com/mne-tools/mne-cpp"><b>https://github.com/mne-tools/mne-cpp</b></a>
 *
 * \section building_sec Building
 *
 * To build the mne libraries <a href="http://qt-project.org/downloads/"><b>Qt5</b></a> is required.
 *
 * Build and install Qt5.
 *
 * To build the whole mne-cpp librarie, do the following steps:
 * Generate the Makefiles using qmake:
 * 	qmake -recursive
 * Make the libraries and examples by running:
 *	make
 *
 * Tip: It's convinient to use the Qt Creator of the <a href="http://qt-project.org/downloads/"><b>QtSDK</b></a> to build the libraries.
 *
 *
 * \section dependencies_sec Dependencies
 *
 * <a href="http://qt-project.org/downloads/"><b>Qt5</b></a>
 * 
 * 
 * \section mailing_list_sec Mailing list
 * 
 * <a href="http://mail.nmr.mgh.harvard.edu/mailman/listinfo/mne_analysis"><b>http://mail.nmr.mgh.harvard.edu/mailman/listinfo/mne_analysis</b></a>
 *
 *
 * \section workflow_contribute_sec Workflow to contribute
 *
 * To contribute to MNE-cpp, first create an account on <a href="http://github.com"><b>github</b></a>.
 * Once this is done, fork the <a href="http://github.com/mne-tools/mne-cpp"><b>mne-cpp repository</b></a>
 * to have you own repository,
 * clone it using 'git clone' on the computers where you want to work. Make
 * your changes in your clone, push them to your github account, test them
 * on several computer, and when you are happy with them, send a pull
 * request to the main repository.
 *
